## Vision
ThingOne is a web-based AI voice assistant built using Django, MongoDB,
and Gemini AI. It supports voice and text interactions through a modern UI.

## Core Features
- Voice & chat interaction
- AI-powered responses
- User-specific memory
- News, weather, and smart commands

## Tech Stack
Backend: Django (Python)
Database: MongoDB
Frontend: HTML, CSS, JavaScript
AI: Google Gemini
